gnome-extension-connman
=======================

gnome-shell extension for connman.

Dependencies
------------

 * gnome-autogen, available from gnome-common
 * [connman-gtk](https://github.com/jgke/connman-gtk)

If building from git, you also need

 * autoconf
 * intltool

Installation
------------

	./autogen.sh
	make
	make install
